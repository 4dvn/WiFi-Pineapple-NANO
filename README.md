# WiFi-Pineapple-NANO-for-China
 
 - 板子尺寸：50mmX85mm。
 - 系统复位键使用的是GPIO15，高电平有效，确定一下实际情况（按手册上做的）。
 - 元件上标NC是代表不焊的意思，加/有数值的代表有后备方案。
